Schedule & Auto-Update

When disabled, devices with firmware newer than the target will be skipped.
When enabled, auto-update will prefer the latest beta release over stable.
Firmware is fetched daily from Tachyon's release pages. Updates roll out gradually: 1 device (canary), then 10%, 50%, 100% — one phase per maintenance window. Any failure pauses the rollout. Weather and time-of-day guards prevent updates during unsafe conditions.
